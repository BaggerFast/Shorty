# Quick Start

1. Download [NodeJS](https://nodejs.org/en/download/)
2. Install bun

```bash
npm i -g bun
```

3. Start up app:

```bash
bun run dev
```

# Docker

1. Download docker
2. Add copy and rename .env.example to .env
3. Fill environment variables
4. Start up docker compose

```bash
docker compose up
```
